# Chat-X 💬

A next-generation messaging app — better than WhatsApp.

## Features
- ⚡ Real-time messaging
- 👥 Group chats
- 📞 Voice & video calls
- 📎 Media sharing
- 🔐 End-to-end encryption
- 🟢 Online/offline status

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
